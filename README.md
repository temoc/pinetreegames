## How to Play Locally

1. Clone this repository:
```bash
git clone https://github.com/temoc/pinetreegames.git
cd pinetreegames
```

2. Open `index.html` in your browser:
```bash
# Windows
start index.html

# Mac
open index.html

# Linux
xdg-open index.html
```

3. Press `SPACE` to start playing!
